Financial Transaction Fraud Dashboard:

Power BI dashboard project focused on analyzing financial transactions to identify potential fraud patterns. The objective was to turn raw transaction data into clear, interactive insights that can help monitor suspicious activities and understand transaction behavior more effectively.
This dashboard allows users to explore transactions by fraud status (fraud vs normal) and card type (credit or debit). It also provides insights into transaction trends over the last 24 hours across different locations, helping to quickly spot unusual patterns. Another key part of the dashboard is the merchant category analysis, which shows how transactions are distributed across categories like restaurants, ATMs, fuel stations, clothing, groceries, and electronics. This helps highlight where most transactions and potential risks may occur.

The dashboard also includes important summary metrics such as:
• previous fraud cases
• weekly and daily transactions
• total merchants and customers
• failed transactions

By bringing all these insights together in one place, the dashboard makes it easier to track transaction activity, monitor fraud trends, and support better data-driven decisions.
